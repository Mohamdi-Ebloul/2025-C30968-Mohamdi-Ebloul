À propos du projet

Mini projet universitaire — Détection du Cancer du Sein par Deep Learning

Contexte académique

Ce projet est un mini projet universitaire développé dans le cadre d'un cours d'Intelligence Artificielle. Il démontre l'application du Deep Learning (CNN) à la classification d'images médicales pour la détection précoce du cancer du sein à partir d'images ultrasonores.

Architecture CNN
Input
224×224×3
Conv2D ×2 + BN
32 filtres, 3×3
Conv2D ×2 + BN
64 filtres, 3×3
Conv2D ×2 + BN
128 filtres, 3×3
GlobalAvgPool → Dense(256) → Dropout(0.5)
Dense(1) — Sigmoid
0 = Benign | 1 = Malignant

Dataset — BUSI

Breast Ultrasound Images Dataset (BUSI) — Al-Dhabyani et al., 2020

  • 780 images d'échographies mammaires
  • 3 classes : Normal, Bénin, Malin
  • Ce projet utilise les classes Bénin / Malin (classification binaire)
  • Disponible sur Kaggle : aryashah2k/breast-ultrasound-images-dataset
Grad-CAM — Explainability

Gradient-weighted Class Activation Mapping (Selvaraju et al., 2017). Génère une carte de chaleur qui met en évidence les régions de l'image qui ont le plus influencé la décision du réseau de neurones, rendant le modèle interprétable pour les cliniciens.

Avertissement : Ce système est développé à des fins éducatives uniquement. Il ne constitue pas un dispositif médical certifié et ne doit pas être utilisé pour des décisions cliniques réelles. Tout diagnostic doit être réalisé par un médecin spécialiste.
