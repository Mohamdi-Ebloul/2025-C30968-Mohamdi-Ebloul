
## Chapitre 6 – Plotting

Visualisation de données scientifiques avec **Matplotlib**

###  Objectifs :
Tracer des données pour l’analyse et la communication des résultats.

###  Outils principaux :
matplotlib.pyplot
mpl_toolkits.mplot3d

### Concepts :
 **Commandes de base** : plot, scatter, hist, bar
 **Personnalisation** :
   Couleurs, styles, légendes, titres
   Axes : set_xlim, set_title, xlabel, etc.
 **Grilles et contours** :
   meshgrid, contour, imshow, contourf
 **Approche orientée objet** :
   fig, ax = plt.subplots()
   Modification d’objets graphiques
 **3D plots** :
   ax.plot_surface(), ax.plot_wireframe()
 **Animation / Films** :
   Enchaînement d’images pour créer des vidéos

## Chapitre 7 – Functions

Utilisation avancée des **fonctions** en Python, outil central de la programmation.

### Concepts importants :
 **Arguments** :
   Positionnels, nommés, par défaut, variables (*args, **kwargs)
 **Retour de valeur** : return
 **Récursivité** : fonction s'appelant elle-même
 **Fonctions anonymes** : lambda
 **Fonctions comme objets** :
   Peuvent être passées en argument, retournées, stockées
   **Closures** et **partielle (functools.partial)**
 **Décorateurs** :
   Fonctions qui modifient d'autres fonctions (@decorator)
 **Documentation** : utilisation de docstrings pour décrire l’usage
